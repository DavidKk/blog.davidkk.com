# WOW 宏

## 通用宏

```
# 钓鱼宏
# 在没进入战斗情况下一键切换 "钓鱼" 装备方案
# 在没进入战斗情况下装备了鱼竿武器使用钓鱼技能
# 进入战斗或点击右键一键切换 "当前" 装备方案
# 进入战斗切换动作条1, 否则切换动作条2

#showtooltip
/changeactionbar [button:2]1;[nocombat]2;1
/equipset [button:2][combat]取消钓鱼
/cast [equipped:鱼竿,nocombat]钓鱼
/equipset [nocombat,noequipped:鱼竿]钓鱼
```

```
# 上马宏
# 在随机上马
# 可游泳状态下使用 "深渊海马"
# 可以飞使用飞行坐骑, 不能飞使用陆地坐骑
# 游泳状态下可飞且没能使用 "深渊海马", 会自动往下执行

#showtooltip
/cast [combat]影遁;[indoorsdwxx]猎豹守护;[swimming]深渊海马
/userandom [flyable]黑色灵翼幼龙,西风幼龙,碧蓝幼龙,暮光幼龙,黑色幼龙,奥的灰烬;化石迅猛龙,炽热战马
```

### 射击

```
# 误导宏
# 在团队或小队中有焦点盟友且没死施放误导
# 没焦点盟友选择目标的目标
# 都没有或者不在小队中选择宠物误导
# 按 "shift" 设置焦点, 也是根据以上来设置

#showtooltip
/focus [@focus,mod:alt,help,raid][@targettarget,mod:alt,help,raid]
/cast [@focus,raid,help,nodead][@targettarget,raid,help,nodead][@pet,help,nodead]误导;[help,nodead][@mouseover]误导;
```

```
# 灵魂治愈与伪装切换宏
# 没进入战斗使用伪装
# 在进入战斗状态使用 "灵龟守护" 或按 "alt" 组合键

#showtooltip
/cast [@player,pet:灵魂兽]灵魂治愈(特殊技能)
/cast [mod:alt]灵龟守护;[nocombat]伪装;[stealth]灵龟守护;[button:2][]灵龟守护
```

```
# 减速宏

#showtooltip

```
